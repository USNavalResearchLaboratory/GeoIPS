GeoIPS is a fully self contained satellite data processing system.

sectorfile:
    package to read in (currently) xml files that contain information regarding areas over which products should be created.

productfile:
    package to read in (currently) xml files that define products for each sensor.

downloader:
    package to ingest data from external sources.

pass_prediction:
    package to determine if and when a satellite overpasses a given sector
